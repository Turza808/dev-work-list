Question-1: What are the different ways to select an element in the DOM?

There are 5 different ways to select an element in the Document Object Model. They are getElementByTagName("tagname"), getElementByClassName("className"), getElementById("id"), querySelectorAll("selector"), querySelector("selector").In the last two method, we have to write the selector with proper symbol(#,.)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : It represent the all html code that an element contains.
innerText : It only represent the text of an element.
textContent : It represent the set of text in an html element and it ignore the html tag and just return the all all text that the element contains.

Question-3: What is event delegation in the DOM?

It's a way to add an event to an element in an HTML document. In this method, we add the event to the parentNode instead of directly on the childNode of that parentNode.

Question-4: What is event bubbling in the DOM?

When an event occurs on the target element, it bubbles up to its parent elements. This propagation is called event bubbling. We can control the propagation by stoping it.

Question-5: How do you create, add, and remove elements using JavaScript?

Create : document.createElement("element");
Add : parentElement.appendChild(childElement);
Remove: parentElement.removeChild(childElement);
